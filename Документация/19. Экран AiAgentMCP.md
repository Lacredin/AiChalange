# Экран AiAgentMCP
Этот файл описывает экран `AiAgentMCP`, предназначенный для проверки MCP-серверов и их инструментов.

## Назначение экрана
`AiAgentMCP` — облегченный операционный экран, где можно проверить подключение к удаленному или локальному MCP-серверу и получить список доступных инструментов.

## Ключевые функции
- Выбор MCP-сервера из набора опций.
- Запрос списка инструментов у выбранного сервера.
- Отображение имени и описания каждого инструмента.
- Показ текущего статуса загрузки и ошибок подключения.
- Быстрое переключение на другие экраны через верхнее меню.

## Зависимости
- MCP-клиент: `RemoteMcpService`.
- DTO инструментов: `McpToolInfo`.
- UI и темы Compose.

## Какие данные использует
- Текущее состояние подключения (`isLoading`, `statusText`, `errorMessage`).
- Список инструментов `tools`.
- Выбранный сервер (`selectedServer`).
- Экран не использует локальную БД для хранения состояния.

## Логика работы
- По команде загрузки выполняется `listAvailableTools` для выбранного endpoint.
- Результат сортируется и отображается списком.
- В случае ошибки пользователь получает текст причины и статус неуспешного запроса.

## Возможности
- Быстрая проверка доступности MCP-инфраструктуры.
- Удобный визуальный контроль перечня инструментов перед использованием в `AiAgentMain`.
- Минимальный набор действий, удобный для диагностики.

## Недостатки и ограничения
- Экран не выполняет полноценные вызовы инструментов в пользовательском потоке.
- Не хранит историю успешных/ошибочных запросов между сессиями.
- Не показывает глубинную диагностику протокольных ошибок, только итоговое сообщение.
